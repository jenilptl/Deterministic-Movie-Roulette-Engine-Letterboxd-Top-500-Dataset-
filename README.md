# Movie Roulette (Letterboxd Top 500 Source)

Production-style full-stack app with:
- strict movie ingestion from Letterboxd official list `https://boxd.it/8HjM`
- direct Letterboxd enrichment (poster, genres, director, cast, IMDb id when available)
- canvas roulette wheel with 500 mapped segments and exact landing animation
- cinematic UI, spin controls, sound effects, and last-5 history

## Project Structure

```
/frontend
/backend
/data
```

## Prerequisites

- Node.js 18+ (Node 20 recommended)

## 1) Install dependencies

```bash
cd backend && npm install
cd ../frontend && npm install
```

## 2) Configure backend env

```bash
cd backend
copy .env.example .env
```

No API key is required for the data build.

## 3) Build strict 500-movie dataset

```bash
cd backend
npm run build:data
```

This script:
1. fetches only `https://letterboxd.com/official/list/letterboxds-top-500-films/` pages 1..5
2. extracts the 500 ranked list entries from that source
3. enriches each entry by scraping the Letterboxd film page
4. validates exact count = 500 with no duplicate slugs
5. writes `data/movies.json`

## 4) Run backend

```bash
cd backend
npm run dev
```

Backend API:
- `GET /api/health`
- `GET /api/movies`

## 5) Run frontend

```bash
cd frontend
npm run dev
```

Open Vite URL (default `http://localhost:5173`).

## Controls

- `Spin Roulette` to spin
- `Spin Again` repeats
- `Space` key triggers spin
- spin button disabled while animation runs

## Notes

- No fallback list is used (source is `https://boxd.it/8HjM`).
- Backend requires `data/movies.json` generated by the strict script.
- Wheel animation uses `requestAnimationFrame`, ease-out deceleration, and deterministic final alignment to the selected movie.
